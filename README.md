# The genetic architecture of the human cerebellar subregions
Code for The genetic architecture of the human cerebellar subregions

## Imaging process

Cerebellum segmentation by CerebNet (https://github.com/panania-entelai/FastSurfer-v2.1.2)

Isolation Forest method in Matlab (https://ww2.mathworks.cn/help/stats/isolationforest.html)

Harmonzationt: Combat Harmonization (https://github.com/Jfortin1/ComBatHarmonization)

Quantile normalization: PLINK (https://www.cog-genomics.org/plink/1.9/)


## Genetic process

Ancestry identification: SNPWeights (https://hsph.harvard.edu/research/price-lab/software/)
Genetic PC calculatation: ABCD_GRM.r (https://github.com/robloughnan/ABCD_GeneticPCs_and_Relatedness)
GRM calculation for CHIMGEN: GCTA-GRM (https://yanglab.westlake.edu.cn/software/gcta/#MakingaGRM)
GRM calculation for ABCD & UKB: GENESIS (https://www.rdocumentation.org/packages/GENESIS/versions/2.2.2))

## Admixed-ancestry GWAS

Autosomal associations for CHIMGEN: LDAK LRM ([https://dougspeed.com/](https://dougspeed.com/single-predictor-analysis/))
Autosomal associations for ABCD & UKB: LDAK-KVIK (http://www.ldak-kvik.com/)
X-chromosal associations: GCTA-fastGWA (https://yanglab.westlake.edu.cn/software/gcta/#fastGWA)

## Single-ancestry GWAS

SNP-based heritability: LDSC (https://github.com/bulik/ldsc)
Cross-ancestry genetic correlation: TAGC (https://github.com/FSSKM/TAGC_review)
Trans-ancestry association mapping: LOGTRAM (https://github.com/YangLabHKUST/LOG-TRAM)
Ancestry inference： Admixture (https://dalexander.github.io/admixture/)
Genotype phasing: SHAPEIT (https://odelaneau.github.io/shapeit5/)
Local ancestry inference: RFmix (https://github.com/slowkoni/rfmix)
Association analyses for AFR: Tractor (https://github.com/Atkinson-Lab/Tractor)

## Fine-mapping analyses

Single-ancesytry fine-mapping analyses: SuSiE (https://github.com/stephenslab/susieR)
Multi-ancestry fine-mapping analyses: MESuSiE (https://github.com/borangao/meSuSie)
Effector gene identification: FLAMES (https://github.com/Marijn-Schipper/FLAMES)

## Genetic clustering and GWAS for genetically informed cerebellar subregions

Genetic correlation analyses: HDL (https://github.com/zhenin/HDL)
Genetic clustering and multivariate GWAS: GenomicSEM (https://github.com/GenomicSEM/GenomicSEM)
multi-variate fine-mapping analyses: mvSuSiE (https://github.com/stephenslab/mvsusieR)
Genetic architecture parameters evaluation: GCTB-SBayeS (https://gctbhub.cloud.edu.au/software/gctb/#Overview)
Stratified LDSC analyses: S-LDSC (https://github.com/bulik/ldsc)
Human accelerated regions enrichment analyses: HARE (https://github.com/ossmith/HARE/)
Cell type association analyses: seismic (https://github.com/ylaboratory/seismic)
PPI analyses: STRING (https://string-db.org/)
Hub gene identification: CytoHubba (https://apps.cytoscape.org/apps/cytohubba)

## Phenotype association analyses

PGS caculation: PRScs (https://github.com/getian107/PRScs)
PHEWAS analyses: PHESANT (https://github.com/MRCIEU/PHESANT)
Genetic correlation analyses: HDL (https://github.com/zhenin/HDL)
Colocalization analyses: coloc (https://github.com/chr1swallace/coloc)
